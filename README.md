# Practice Player

YouTubeを見ながらギター練習するための、シンプルな練習用プレイヤーです。

## 主な機能

- A-Bループ
- 再生速度変更
- 区間保存
- ブックマーク
- TAP BPM
- 履歴保存

## できること

気になるフレーズだけをA-Bループで繰り返したり、速度を少しずつ落として練習したり、難所を区間保存してあとで呼び出したりできます。  
ギターのコピー、フレーズ練習、リズム確認などを、できるだけ直感的に行えるように作っています。

## 使い方

1. YouTube URLを貼り付ける
2. 動画を読み込む
3. 必要に応じてA点・B点を設定する
4. 速度変更やループを使って練習する
5. 必要なら区間保存やブックマークを使う

## 対応機能

- YouTube動画の読み込み
- 再生 / 一時停止
- シーク
- A-Bループ
- 再生速度プリセット
- 微調整速度変更
- 区間保存
- 保存区間の再読込
- 保存区間の進捗管理
- ブックマーク保存
- TAP BPM

## 注意

- 一部のYouTube動画は埋め込み制限により簡易モードになる場合があります
- ブラウザや動画側の制限により、一部挙動が異なることがあります
